# Quick Start Guide

## Installation Steps

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Available Pages

- **Homepage**: `/` - Main landing page with hero, collections, and testimonials
- **Products**: `/products` - Product listing with filters
- **Product Detail**: `/products/1` - Individual product page
- **Checkout**: `/checkout` - Multi-step checkout flow
- **Admin Dashboard**: `/admin` - Admin control panel with AI insights

## Features to Explore

1. **AI Assistant Orb** - Click the floating orb in the bottom-right corner
2. **Product Cards** - Hover to see animations and quick actions
3. **Navigation** - Scroll to see glassmorphism effect
4. **Mobile View** - Resize browser to see mobile bottom navigation
5. **Admin Dashboard** - View AI agent logs and recommendations

## Design Highlights

- Glassmorphism effects throughout
- Smooth animations with Framer Motion
- Responsive design for all screen sizes
- Futuristic gradient accents
- Premium typography scale

## Troubleshooting

If you encounter TypeScript errors:
1. Make sure all dependencies are installed: `npm install`
2. Restart your development server
3. Clear `.next` folder if needed: `rm -rf .next`
